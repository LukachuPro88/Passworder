# Passworder

Passworder is a Python library providing utilities for password generation, strength checking, masked password input, hashing, and password verification. Passworder is currently at version **0.0.3**.

## âœ¨ Features

* **Generate random passwords**:

  * **Uppercase / lowercase / symbol combinations** using (`string_password`)
  * **Numbers only** using (`number_password`)
  * **Fully customizable strong passwords** using (`password`)
* **Password strength evaluation** using (`password_strength`)
* **Terminal masked password input** using (`password_input`)
* **Hash passwords and save them to a file** using (`hash_password`)
* **Check if a password exists in a hashed file** using (`find_password`)
* **Auto-copy passwords to clipboard** using `pyperclip`
